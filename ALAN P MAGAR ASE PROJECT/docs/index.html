How to run

  1. Open Booseapp.sln in Visual Studio.
  2. Set startup project to booseapp.
  3. Run to execute BOOSE programs and draw on the canvas.

Key features implemented

  • Drawing: moveto, drawto, circle, rect, pen/pencolour, clear, reset, set
  • Variables: int, real, boolean, string
  • Arrays: array + poke/peek
  • Control flow: if/else, while, for
  • Methods: method + call
  • Unit tests: MSTest (see docs/test-cases.md)
